# Sequential TCN
## Usage
For more details, check:
```shell
python setup.py -h
```
### Sequential MNIST Task
* GPU version:
```shell
python setup.py --cuda
```
* CPU version:
```shell
python setup.py
```
### Sequential P-MNIST Task
P-MNIST task means permuted MNIST task.
* GPU version
```shell
python setup.py -p --cuda
```
* CPU version
```shell
python setup.py -p
```
## Requirements
* python3.5+
* [pytorch](https://pytorch.org/)
